1. Пользователь запрашивает свободные места на рейсе при помощи FreeSeats. Возвращается список свободных мест List(SeatNo)

	Имея список свободных мест пользователь может вызывать остальные функции

2. Пользователь может анонимно купить место на рейсе при помощи BuyFree.
3. Авторизованный пользователь может забронировать место на рейсе при помощи Reserve.
4. Авторизованный пользователь может продлить свою бронь на 3 дня ExtendReservation.
5. Авторизованный пользователь может выкупить забронированное место BuyReserved.
